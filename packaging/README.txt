Kura5 Savehelper version 0.1-dummy-alpha

This version is only for testing purposes. It does not
hold any code able to interact with save files. Feel free
to abuse the code as much as you want. If you encounter
any issues, please report them on the GitHub page.

https://github.com/EarthlySkies/kura5-savehelper

Installation:
1. Extract the contents to your Kura5 game directory
2. Move the "savehelper.py" out of the "savehelper.d" directory
3. Launch the helper from your file explorer or terminal
4. The program is now running and will ask you what you want to do
